# Як вставити багатосторінковий сайт у Google Sites

Цей проєкт містить кілька сторінок:
- `index.html`
- `about-ai.html`
- `technologies.html`
- `ethics.html`
- `project-plan.html`
- `styles.css`
- `script.js`

## 1) Публікація на GitHub Pages
1. Створіть новий репозиторій на GitHub.
2. Завантажте **усі файли** з цього проєкту в корінь репозиторію.
3. Відкрийте: **Settings → Pages**.
4. Оберіть:
   - **Source:** Deploy from a branch
   - **Branch:** `main` + `/root`
5. Збережіть. Через 1-2 хв з'явиться URL виду:
   `https://<username>.github.io/<repo-name>/`

## 2) Вставка в Google Sites
1. Відкрийте Google Sites.
2. Натисніть **Вставити → Вбудувати**.
3. Вкладка **За URL** → вставте посилання GitHub Pages.
4. Натисніть **Вставити**.
5. Розтягніть блок по ширині, опублікуйте сайт.

## 3) Рекомендації
- Для стабільної роботи навігації не перейменовуйте html-файли.
- Якщо блок обрізається у Google Sites, збільште висоту вбудованого елемента.
- Перевірте відкриття всіх підсторінок уже після публікації на GitHub Pages.
