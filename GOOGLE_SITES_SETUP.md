# Як завантажити цей сайт на GitHub і вставити в Google Sites

## Файли сайту
- `index.html`
- `history-ai.html`
- `ai-usage.html`
- `smartphones-social.html`
- `jobs-risk.html`
- `ai-games.html`
- `project-plan.html`
- `styles.css`
- `script.js`

## 1) Публікація на GitHub Pages
1. Створіть репозиторій на GitHub.
2. Додайте всі файли з цього проєкту в корінь репозиторію.
3. Відкрийте **Settings → Pages**.
4. Увімкніть:
   - **Source:** Deploy from a branch
   - **Branch:** `main` і папка `/root`
5. Збережіть налаштування.
6. Отримайте посилання виду:
   `https://username.github.io/repository-name/`

## 2) Вставка в Google Sites
1. Відкрийте ваш Google Sites.
2. Оберіть **Вставити → Вбудувати → За URL**.
3. Вставте посилання на GitHub Pages.
4. Натисніть **Вставити**.
5. Розтягніть блок, щоб сайт займав більше місця.
6. Опублікуйте Google Sites.

## 3) Корисні поради
- Не перейменовуйте файли після публікації, щоб не зламати навігацію.
- Перевірте відкриття кожної підсторінки.
- Якщо блок в Google Sites обрізається, збільшіть його висоту.
